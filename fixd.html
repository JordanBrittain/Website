A list menu

What?

Originally, the FIXD application was navigated using both a tab bar and hamburger menu. The new app is navigated via only a list menu.

 

Why?

Itâ€™s important to understand the context in which people will be using this app. It is a tool. People will go into it with a clear task already in mind, whether that be to scan their engine, log some maintenance work, etc. Itâ€™s important that theyâ€™re able to intuitively navigate to the function they need. A list menu is ideal for this sort of use, as it provides the user with a simple and obvious path to their desired function. It does not let the user navigate between functionality as quickly as a tab bar would, but thatâ€™s not as important being that users will be coming into the app with a single task and using one branch of functionality in most scenarios of use. A clear and obvious navigation system is paramount, and a list menu provides this.

A Consolidated Timeline

What?

The FIXD app allows users to see their vehicleâ€™s recommended maintenance schedule, log any maintenance they get done on their vehicle, and keeps a history of all the engine scans performed on the vehicle. Originally, this info was stored in different places. In the new app, all of this info is consolidated into one single timeline.

 

Why?

There wasnâ€™t much of a reason to have all of this info in separate locations in the first place. If a huge amount of info was being logged, then this would make sense as it would split it up and make it more discernible. But this isn't the case. For most users, there will be only a dozen or so logs over the course of a year. In order to make the app more user friendly and given that these logged items are correlated, consolidating all of this info into one timeline is the appropriate choice.

The typeface Industry was used for the larger copy while Open sans was used for the smaller stuff. Industryâ€™s typeforms follow a rigid structure, reinforcing the reliable tone. Open sans is very versatile and designed to be readable on screens at small sizes.
